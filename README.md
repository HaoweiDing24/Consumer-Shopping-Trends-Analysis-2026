
# Consumer Shopping Trends Analysis 2026
ACC102 Mini Assignment - Data Product

## 1. Project Purpose
This project analyzes consumer shopping behavior, including age, income, online/offline spending, and shopping preferences.

## 2. Dataset
- File: Consumer_Shopping_Trends_2026.csv
- Contains customer age, income, spending habits, preferences, etc.

## 3. Files in this Repository
- shopping_analysis.ipynb (main analysis code)
- README.md (project introduction)

## 4. Analysis & Visualizations
- Age distribution
- Shopping preference (Online/Store/Hybrid)
- Monthly income vs online spending
- City tier vs shopping preference
- Online vs offline spending comparison
- Consumer behavior correlation heatmap

## 5. Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
